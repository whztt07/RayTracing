RayTracing
Offline CPU && GPU ray tracer, real time CUDA ray tracer.

Original project(SVN) by codeboycjy: https://cudart.svn.sourceforge.net/svnroot/cudart 
You can find more info in his blog: http://blog.csdn.net/codeboycjy/article/details/6525465 (chinese)


Requirement:
	D3D9 (DXSDK_Jun10 http://www.microsoft.com/en-us/download/details.aspx?id=6812)
	CUDA
